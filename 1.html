HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup Language for creating web pages and web applications.

Web browers receive HTML documents form a web server or form local storage and render them into multimedia web pages.
HTML은 섬세하다 웹페이지의 심플함과 오리지날적인 단서가 포함되어 있기 때문이다. 서류에 적용을 하기 위해서.
HTML요소는 HTML페이지 블록을 만든다. html구조와 함께 이미지와 다른 객체 즉, 직관적인 형태와 아마도 프론트 페이지 안을 포함하는 것과 같이. 그것은 창조적인 구조와 섬세성을 포함한다.
